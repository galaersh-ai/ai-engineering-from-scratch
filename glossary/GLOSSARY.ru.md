## Правила перевода терминов

1. **Устоявшийся перевод есть** → используем его (gradient → градиент, layer → слой)
2. **Нет устоявшегося** → транслитерация (token → токен, embedding → эмбеддинг)
3. **Аббревиатуры** → без перевода (LLM, GPU, RLHF, RAG, CUDA)
4. **Названия библиотек** → без перевода (PyTorch, NumPy, TensorFlow)
5. **Глаголы** → адаптируем (to train → обучать, to infer → выполнять инференс)
